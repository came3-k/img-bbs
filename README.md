##掲示板
「画像投稿掲示板」です。
画像(jpg,png)+テキストを投稿し、それに対する評価を付けることが出来る仕様です。

> 金子達哉『pixivエンジニアが教えるプログラミング入門』星海社新書,2015

を参考にして作成しました。

####以下の3点を練習しながら学習するために行った。

* Ruby
* データベース(今回はSQLite)
